== Description ==

Profile Builder - Custom Profile Menus Add-On


Add custom menu items like Login/Logout or just Logout button and Login/Register/Edit Profile in iFrame.
Hide added items based on user role or choose to hide only for logged in / logged out users.


== Installation ==

1. Upload and install the zip file via the built in WordPress plugin installer.
2. Activate the plugin from the "Plugins" admin panel using the "Activate" link.
3. Go to Appearance -> Menus -> PB Custom Profile Menus to add custom menu items.
